Notas RM

Uma mão na roda na hora de cadastrar as notas.

Instalação

  1. Habilite a barra de favoritos do seu navegador
  2. Arraste cada um dos três links abaixo para a barra de favoritos
  3. Acesse o RM na página de inserção de notas
  4. Clique no ítem Habilitar NotasRM para habilitar a ferramenta
  5. Utilize o ítem Com Todos para definir uma nota para todos os alunos
